Prototype d’équipe

Montage 11 + banc · cartes issues du catalogue FC26 (Données jeu base roster, pas chaque variante FUT promo)

Important : le fichier data/fc26-cards-catalog.json agrège le pool public EAFC26-DataHub (notes / attributs alignés FC26). Ce ne sont pas toutes les cartes spéciales Ultimate Team (TOTY, événements, versions IF) — EA ne publie pas ce fichier en open data. Pour un prototype tactique (postes, profils), le catalogue de base suffit ; les promos restent à saisir manuellement ailleurs si besoin. Chargement du JSON : ouvrez l’app via un serveur HTTP local (ex. python3 -m http.server à la racine du dépôt) : le protocole file:// bloque souvent fetch du catalogue.

Méthodologie & scores

Transparence · limites

Meta score joueur (app)
Formule démo

Pondération indicative : pace utile 20 %, finition 15 %, passe 15 %, défense 10 %, physique 10 %, polyvalence poste 10 %, playstyles 10 %, stabilité patch 5 %, valeur/cost 5 %. Données EA item non présentes → score non fiable.

metaScore = clamp( sum(weight_i * norm(metric_i)), 0, 99)
Meta score formation
Formule démo

Équilibre déf/att, création, densité axe, couverture, exécution, adaptabilité, résistance patch (heuristique analyste), synergie profils — normalisé sur 100 dans l’UI.

Meta score technique
Formule démo

efficacité estimée − coût difficulté − risque interception + bonus contexte (ex. ligne haute) ; patch sensitivity flag manuel.

SBC score
Calcul local

Aucune API EA pour EV pack — tout chiffre marché = externe. Afficher méthodologie et date marché.

Mise à jour après patch

Workflow mainteneur

1. Lire la source EA (news ou forums). 2. Dupliquer le template dans fc26-meta-data.js → PATCHES. 3. Mettre à jour CONFIG.lastOfficialPatchId + compteurs. 4. Ajuster META_SNAPSHOT, BEHAVIORS, patchSens joueurs concernés. 5. Exporter backup JSON ci-dessous.
Check-list express
Démarche
  • Copier les lignes « gameplay / UT » officielles mot pour mot si possible.
  • Ne pas changer les badges : tout ce qui n’est pas EA reste communautaire/analytique.
  • Re-trier SBC après maj prix FUTBIN (champs cost / reward).
  • Vérifier que le fichier data est bien dans le même dossier que le HTML.
